# ChristBot

## Instalasi awal

1. Update package

```bash
pkg update && pkg upgrade
```

2. Install nodejs

```bash
pkg install nodejs
```

3. Install git

```bash
pkg install git
```
4.install yarn

```bash
pkg install yarn
```

5. Cloning repo

```bash
git clone https://github.com/Novan-H/ChristBot
```

## Cara Pasang

```bash
cd ChristBot
```
```bash
yarn install
```

2. Setelah selesai menginstal, jalankan bot dengan perintah:

```bash
npm start
```
